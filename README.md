## 📖 About

Smart Agriculture System is a Full Stack MERN application that helps farmers manage their profiles and access smart agriculture features such as weather information, crop recommendations, and dashboard management.

## ✨ Features

- 🔐 User Login & Signup
- 👤 Profile Management
- 📊 Farmer Dashboard
- 🌦 Weather Information
- 🌱 Crop Recommendation
- 📩 Contact Page
- 🔥 Firebase Authentication
- ☁ MongoDB Atlas Database
- 🚀 Railway Backend Deployment
- 🌐 Vercel Frontend Deployment

## 🛠 Tech Stack

- React.js
- Vite
- Node.js
- Express.js
- MongoDB Atlas
