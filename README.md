# 🌾 Smart Agriculture System

A full-stack Smart Agriculture web application built using the MERN Stack to help farmers manage their profiles, check weather conditions, and receive crop recommendations.

## 🚀 Live Demo

🔗 https://smart-agriculture-system-phi.vercel.app

---

# ✨ Features

- 🔐 User Authentication (Login & Signup)
- 👤 Profile Management
- 🌦️ Live Weather Information
- 🌱 Crop Recommendation
- 🚜 Farmer Dashboard
- 📞 Contact Page
- ☁️ MongoDB Atlas Database
- 📱 Responsive Design

---

# 🛠 Tech Stack

### Frontend
- React.js
- Vite
- CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend : Vercel
- Backend : Railway

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ankitsinghdhakar537-web/smart-agriculture-system.git
```

### Go to Project Folder

```bash
cd smart-agriculture-system
```

### Install Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
cd backend
npm install
npm start
```
